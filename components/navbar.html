Navbar

A navbar is the top navigation bar that provides global wayfinding, user and account controls, and search entry. It should remain consistent and compact across Company web products.

Anatomy (left → right)

  1. Brand (logo)
  2. Primary sections: Home, Workspace, My Files, Projects, All Users Files, Activity
  3. Global search (leading icon)
  4. System actions: Notifications, Settings
  5. User menu (avatar)
  6. Window controls (minimize, restore, close) — desktop wrapper only, not in web

Layout and spacing

Property Value
Navbar height 56px (desktop wrapper and tablet)
Horizontal padding 16px at ≤1440px; 24px beyond
Item spacing 1.5rem between items; 0.25rem icon–label gap
Logo max height 28px
Search input width 250–350px default, expands on focus

Conflict noted: older specs list 350px in Layout and 250px in Search — reconcile when implementing in product.

States

  • Default: secondary text color
  • Hover: background tint, pointer cursor
  • Active/current: underlined primary section

Responsive behavior

  • ≥1024px: full labels, expanded search
  • <1024px: icons only for menu items

Interactions and content

  • Dropdowns: click to open; Escape or click outside closes.
  • Short labels (1–2 words), nouns not verbs; avoid obscure abbreviations.
