Dropdowns

A dropdown is a UI control that presents a compact list for choosing one option or executing actions. This page covers both selection pickers and hierarchical menus.

Selection dropdown (standard)

Single option from a predefined set (country, font, category). Fit the maximum useful options without overcrowding; use scroll or grouping for long lists.

Action menu

List of actions or navigation options. Use dividing lines between groups, not full borders.

Multi-level dropdown

Nested submenus for complex navigation or settings. Show hierarchy with indent, arrows, or icons. Consider back navigation or breadcrumbs for deep trees.

UX tips

  • Group options by category, relevance, or A–Z.
  • Use dividers or indent for hierarchy.
  • Add search or filter for long or nested lists.
  • Menu items use sentence case.
  • Hover highlight indicates interaction.
  • Use → for internal navigation and ↗ for external links when applicable.
