Forms

Forms are UI patterns for collecting user input with consistent spacing, typography, and focus states. This page documents text fields, labels, validation, and related controls.

Elements

Element Notes
Field wrapper Vertical spacing between fields
Label Sentence case; sits above the control; disable label styling when the control is disabled (checkbox/radio only)
Text input 28px min-height, focus ring
Select Same sizing and states as input; see Dropdowns for selection menus

Guidelines

  • Labels use sentence case and sit above controls.
  • Group related fields with consistent 6px spacing between controls.
  • Focus states use border color + subtle box shadow — never remove focus indicators.
  • Show inline validation on blur or submit with specific, actionable messages.
  • Related options use fieldset and legend (see Checkboxes).
