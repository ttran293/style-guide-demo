Animation and Motion

Animation and motion are UI feedback patterns that guide attention and communicate state changes. Every animation must be purposeful, consistent, and accessible.

Durations

Name Value Use case
Fast 150ms Focus states, quick feedback, press
Normal 200ms Hover states, micro-interactions
Slow 300ms State changes, fade in/out
Slower 500ms Validation shake, emphasis

Easing

Name CSS value Use case
In-out cubic-bezier(0.4, 0, 0.2, 1) Most interactions, state changes (300ms)
Out cubic-bezier(0, 0, 0.2, 1) Hover states, micro-interactions (200ms)
In cubic-bezier(0.4, 0, 1, 1) Focus states, quick feedback (150ms)
Linear linear Progress indicators, loading

Reduced motion

Honor prefers-reduced-motion: reduce. When enabled, animation and transition durations collapse to near-instant values so motion-sensitive users are not affected.

  • Feedback — confirm user actions and system responses
  • Guidance — direct attention to important elements
  • State changes — communicate transitions between states
  • Seizure safety — avoid flashing or rapidly changing content
