Icons

Icons are UI elements that express a single concept in ways people instantly understand. Choose solid or outline styles based on recognition cues and context.

Icon sizes

Size Usage
16px Inline with text, compact controls
20px Buttons, menu items

Solid vs outline

Solid — faster recognition for physical objects; use when outline creates narrow inner spacing.

Outline — better when characteristic cues sit on outer edges (comment bubble, trash can, key).

When to use each style

Solid Outline
Physical, real-world objects Wide inner spacing in the shape
Highlight a particular color Subtle cues along outer edges
Narrow inner spacing in outline version Simple designs with no inner elements
